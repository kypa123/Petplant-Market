# Petplant-Market

jsp, servlet으로 이루어진 반응형 웹 쇼핑몰입니다.
부트스트랩을 사용했기에 모바일 뷰를 지원합니다.

로그인 기능, 다국어처리, 예외처리, 기본적인 시큐리티를 구현하였습니다.
